# Criterios de Diseño Geométrico basados en TPDA

*Nota: Estos criterios reciben el TPDA como parámetro de entrada para definir geometría y seguridad, pero NO forman parte del motor matemático de cálculo o expansión del TPDA.*

## Anchos de Berma
- **Regla:** El ancho de las bermas depende del tráfico.
  - Para TPDA > 750: berma de 1.5 m.
  - Para TPDA < 750: berma de 1.0 m.
- **Fuente:** S24-SCV-MA-4_DISEÑO DE CARRETERAS.pdf, Pág. 151.

## Barreras de Contención (Seguridad)
- **Regla:** Para determinar la necesidad de barreras en base al ábaco de altura de terraplén vs tráfico, en el caso de **calzadas unidireccionales**, se debe utilizar el **50% del TPDA total de la ruta**.
  *(Nota: Esto aplica únicamente a este diagrama de seguridad geométrica, y no es un factor de distribución direccional para cálculo de ESAL).*
- **Fuente:** S24-SCV-MA-4_DISEÑO DE CARRETERAS.pdf, Pág. 167.

## Clasificación de Túneles
- **Regla:** Los proyectos de túneles en vías de bajo tránsito permiten consideraciones especiales si el TPDA < 2500 (vehículos equivalentes/diarios).
- **Fuente:** S24-SCV-MA-4_DISEÑO DE CARRETERAS.pdf, Pág. 200.
