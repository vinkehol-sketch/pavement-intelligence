# Correspondencia Clasificación ABC y Configuraciones de Ejes

El sistema permite asociar las 13 categorías vehiculares de la ABC con posibles configuraciones de ejes. Un vehículo de una categoría puede presentar múltiples configuraciones.

| Categoría ABC | Descripción | Configuración Típica Principal | Otras Posibles |
|---------------|-------------|--------------------------------|----------------|
| 1 | Automóviles, Vagonetas | 1RS-1RS | - |
| 2 | Camionetas | 1RS-1RS | - |
| 3 | Minibuses | 1RS-1RS | - |
| 4 | Microbuses (2 ejes) | 1RS-1RS | 1RS-1RD |
| 5 | Buses Medianos (2 ejes) | 1RS-1RD | - |
| 6 | Buses Grandes (3+ ejes) | 1RS-2RD | 1RS-1RD-1RD |
| 7 | Camión Mediano (2 ejes) | 1RS-1RD | 1RS-1RS |
| 8 | Camión Grande (2 ejes) | 1RS-1RD | - |
| 9 | Camión Grande (3 ejes) | 1RS-2RD | 1RS-1RD-1RD |
| 10 | Camión Semirremolque | 1RS-2RD-2RD / 1RS-2RD-3RD | Múltiples combinaciones |
| 11 | Camión Remolque | 1RS-1RD-1RD-2RD | Múltiples combinaciones |
| 12 | Motocicletas | 1RS-1RS | - |
| 13 | Otros (Agrícolas) | Variable | Variable |

*Atención*: La plataforma no asumirá una configuración rígida sin verificar los datos del pesaje o aforo.
